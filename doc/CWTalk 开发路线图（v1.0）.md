## 阶段 0：Qt 热身（1-2 周）

**目标：** 熟悉 Qt 基础，建立开发环境

| 任务                  | 产出                        | 学习点                       |
| --------------------- | --------------------------- | ---------------------------- |
| 搭建 Qt 环境          | 可编译的空项目              | Qt Creator, qmake/cmake      |
| 写一个小串口工具      | 能打开 COM 口，收发字符串   | `QSerialPort`, 信号槽        |
| 写一个 Grid 计算器    | 输入两个 Grid，算距离和方位 | 字符串处理， Maidenhead 算法 |
| 读一本 Qt 入门书/教程 | 笔记                        | 信号槽、布局、事件、线程     |

**验收标准：** 能独立写出带 GUI 的串口调试助手（收发文本）

## 技术栈建议

| 组件         | 建议                                              |
| ------------ | ------------------------------------------------- |
| **Qt 版本**  | Qt 5.15.2 LTS（稳定，资料多）或 Qt 6.5+（新特性） |
| **构建工具** | CMake（跨平台，现代）                             |
| **串口**     | `QtSerialPort` 模块                               |
| **配置存储** | `QSettings`（INI 格式）或 JSON                    |
| **线程**     | `QThread` + 信号槽（避免裸线程）                  |
| **定时器**   | `QTimer`（高精度用 `QElapsedTimer` 校准）         |

## 第一个冲刺建议

2. 同时找一个 CTY 数据文件（如 `cty.plist` 或 `cty.dat`），写解析代码
